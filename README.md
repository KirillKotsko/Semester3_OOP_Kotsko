Бібліотека: QCustomPlot (qcustomplot.h, qcustomplot.cpp)

Програма: ілюстрація графіків деяких функцій з можливістю змінювати значення коефіцієнтів цих функцій.

Була обрана саме ця бібліотека, адже з її допомогою можна намалювати будь-який графік функції, і не тільки. Після прочитання документації та туторіалів одразу ж стало зрозуміло, як встановлювати та використовувати цю бібліотеку. Посилання, які допомагали при створенні програми:
